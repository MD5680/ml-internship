# Disease Outcome Prediction Using Machine Learning

## Project Description
This project is part of my departmental internship. The objective is to predict disease outcomes (Positive/Negative) based on patient symptoms and clinical attributes using machine learning techniques.

## Dataset Information
- Domain: Healthcare
- Records: 349
- Features: Symptoms, Age, Gender, Blood Pressure, Cholesterol Level
- Target Variable: Outcome Variable

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Excel
- Kaggle Dataset

## Methodology
1. Data collection from Kaggle
2. Data cleaning and preprocessing (Excel + Python)
3. Exploratory Data Analysis (EDA)
4. Feature encoding and scaling
5. Model training and evaluation
6. Model comparison and result analysis

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- Naive Bayes

## Results
Random Forest achieved the highest accuracy among all models.
